Scrape entries from KIT Open

Our Scraper is Python code that uses the Selenium web scraping library to scrape publications from the KIT Open website.
The code navigates to the website, sets filters, and then scrapes the information for each publication. It saves the scraped data in a CSV file, with the structure defined in the "HEADER" variable.

The code uses the Chrome browser, which can be run in headless mode (without the GUI) if the "HEADLESS" variable is set to True. The Chrome Driver and the Chrome browser executable must be specified in the "DRIVER_PATH" and "CHROME_PATH" variables, respectively.

The output file path is specified in the "OUTPUT_PATH" variable, and the number of results to be scraped is defined in the "NUMBER_OF_RESULTS" variable.

In Google Chrome, there are several ways to view the XPaths of elements on a website. We recommend to just use the developer tools: You can open Chrome's developer tools by clicking the "F12" button or by going to the browser menu and selecting "Developer" and then "Developer tools". Then click on the "Elements" icon and select the element whose XPath you want to view. The XPath of the element will be displayed in the lower section of the developer tools.
