This is Python code that uses the Selenium web scraping library to scrape publications from the KIT Open website (https://primo.bibliothek.kit.edu/primo-explore/search?tab=kit_evastar&sortby=date&vid=KIT). The code navigates to the website, sets filters, and then scrapes the information for each publication. It saves the scraped data in a CSV file, with the structure defined in the "HEADER" variable. The code uses the Chrome browser, which can be run in headless mode (without the GUI) if the "HEADLESS" variable is set to True. The Chrome Driver and the Chrome browser executable must be specified in the "DRIVER_PATH" and "CHROME_PATH" variables, respectively. The output file path is specified in the "OUTPUT_PATH" variable, and the number of results to be scraped is defined in the "NUMBER_OF_RESULTS"Â variable.
